## 🛠️ Building

### Prerequisites

- Flutter SDK (3.24.5 or higher)
- For iOS/macOS: Xcode
- For Android: Android Studio
- For Windows: Visual Studio
- For Linux: Required system dependencies:
  ```bash
  sudo apt install libmpv-dev mpv
  ```
